# Murder Mystery AI — Going Live Checklist

## Overview
Multiplayer 1920s noir mystery game. Monetise via alphinium-games-multiplayer subscriptions, event licensing, and corporate team-building packages.

## Step 1: Real Multiplayer Backend (~1 day)
1. Connect alphinium-games-multiplayer addon:
   - Replace simulated `demoPlayers` joining with real WebSocket room logic
   - Room codes become real server-managed sessions
   - Phase transitions broadcast to all connected players
2. Each player sees their own character (not the host's Detective view)
3. Game state syncs: clue reveals, accusations, reveal screen — all players see simultaneously

## Step 2: Additional Scenarios
Create a scenario library (each is a separate JSON config):
- `mansion.json` — current "Murder at Midnight Manor"
- `yacht.json` — "Death on the Adriatic" (maritime thriller)
- `casino.json` — "The Monte Carlo Affair" (1960s heist)
- `space.json` — "Station 7: Zero Gravity" (sci-fi murder)
Load scenarios from alphinium-cms or a simple hosted JSON API.

## Step 3: Real AI Voices (alphinium-ai)
1. Each character voice → ElevenLabs voice clone or pre-set persona
2. Clue narration read aloud by the narrator voice
3. Character responses generated by LLM in-character
4. Host can trigger "Ask the character a question" — real AI response in voice

## Step 4: Monetisation Options
| Model | Details |
|---|---|
| Pay-per-game | $2.99/room/game — Stripe via alphinium-payments |
| Event package | $49/event (unlimited players, 3 scenarios) |
| Corporate team-building | $199/session — custom branding + facilitator mode |
| Subscription | $9.99/mo — unlimited games, all scenarios |

## Step 5: Deploy
1. `expo export --platform web` → Netlify/Vercel
2. App stores: `expo build:ios` + `expo build:android`
3. Host on `mystery.alphinium.com` or white-label

## Marketing Angle
- Corporate team-building is the highest LTV customer
- Hen nights, birthday parties, date nights are high volume
- Partner with event venues, escape room operators
